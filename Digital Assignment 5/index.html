Ninja Guess by Joseph Aversa

  • Originality: Ninja Guess is 2D non-jumping platformer that focuses more on minor gambling than actual action. The game is singleplayer, it can also be used to for high score competition (getting the highest score in Donkey Kong).
  • Technical Merit: While the game itself is very simple, it does implement several interesting technological tricks to contextualize the game mechanics. The ninja character utilizes an old-fashion sprite-sheet animation as he moves left or right. The game also utilizes if/else statements to trigger events, such as picking up an object to display a message on screen.
  • Prototype Postmortem: Even as you learn new techniques in game development, there are still limits to what you can and can't do. The game was originally going to be a series of math questions to answer. The game was failing to do the if else statements correctly in the update function to change between different questions. I had to refer back to Ninja Guess's original concept as a random number generator. I still struggled to get the physics correct between the character and the platform. Resetting the game, however, became very simple to accomplish.

  • Prototype Closest Other Game: The game is an improvement from Ninja Guess in Digital Assignment 1. Ninja Guess now has several states, consisting of Boot, Preloader, MainMenu, NinjaGuess, and GameOver. With the addition of states, the player will no longer need to press the refresh button on the top of the webpage to restart the game. In the last game, the player had to guess one number to win. In this game, the player will need to guess multiple times, until he or she answer incorrectly to earn a high score. Instead of running into the number boxes to guess a number, the player will now walk over the number box, and press attack to guess the number and play an attack animation. The game also implements sound effects at different events.

    Like the original game, this game plays similarly to NES titles like "Ninja Gaiden" and "Castlevania," but with similarities to mid-90's edutainment games like "Mario Teaches Typing". The main objective of the game as an 80's mentality, where there is a focus for getting a high score, instead of completing a story.

  • High Concept: Guess a number between 0 and 1 as many times as you can.
  • Theme: The themes implemented in this game include a mix of 90's point and click, and classic NES.
  • Mandated Variety: This 2D platformer is casually played using only the A,D,F keys on specific rules.
  • Prototype Goal: To improve upon an old prototype with the better game development knowledge.
  • Player Experience Goals: The player will have a short, pieceful, yet entertaining time as they compete with themselves (and others) in guessing the most amount of times.
  • Gameplay: The player begins with a Menu Screen with a control set up. He or she must use the cursor for the start button to start the game. The game begins with a ninja character and a '1' and '0' box inbetween him. The objective of the game will be at the top of the screen, and a number count at the center. The player must press 'A' to move left, 'D' to move right, and 'F' to attack/select. The player must attack one of the number boxes to answer the guess. If the player answers correctly, the number count will increase by one and a new randow number between 0 and 1 is generated. The player will do this until he/she answers incorrectly. If the player answers incorrectly, a Game Over Screen will appear with an amusing animation, the high score, and a start button to return back to the Main Menu.
  • Strategies: Because it is a guessing game, there will be more gambling involved than actual strategy.
  • Story/Setting/Premise: The young, nameless ninja must master his training to become a ninja master. To be an excellent ninja, he must be a quick thinker. He is given a task from the heavens for his training: answer different guesses multiple times without fail. He must continue this guessing game until he becomes a master.
  • Target Audience: People under all ages can enjoy this game, specifically people who enjoy earning the highest record in any game.
  • Play Time: Each game will most likely take 5 seconds to play. But the pure enjoyment of the game will have the player play multiple times, creating longer play time.
