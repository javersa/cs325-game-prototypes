Heaven's Surprise

  • Originality: The game is a top-down 2D game based off of Game Maker indie titles like Yume Nikki.
  • Technical Merit: The game utilizes 2D sprite animations for horizontal and vertical movements.
  • Prototype Postmortem: Learning the next level in game development and making smaller games to make a bigger is a challenge. It was a challenge to load different assets for different portions of the game. Because of this barrier, I was unable to make the game larger with different schenarios. I want the game to be more of a dungeon crawler similar to Legend of Zelda: A Link to the Past and Diablo, where you go around different parts of the world to performing different tasks to collect keys to win.

  • Prototype Closest Other Game: The game mechanics are similar to Yume Nikki: you are simply moving up, down, left, and right; pick up items; and enter doors.
    • The code for this example is adapted from the Phaser Move A Sprite
  • High Concept: A brave warrior arrives to heaven and must collect the key to enter the door.
  • Theme: The game required a man with wings to take advantage of the heaven influences.
  • Mandated Variety: The game requires mouse input, A,S,W,D. The game has no randomness (except a surprise at the end) in a top-down adventure game. The game requires certain inputs to trigger events.
  • Prototype Goal: The game will take advantage of the states in Phaser.
  • Player Experience Goals: The game is not meant to be played multiple times, for it is more visually entertaining than interactively entertaining.
  • Gameplay: The game loads to a Start Menu with an Angel flying around the screen. The player presses the start button to play the game with a left mouse click. The player will play a Warrior arriving to heaven. He is approached by an Angel, giving him the task to find the Key of Destiny if he wants to enter heaven. The Warrior moves up, down, left, and right with W, S, A, D. The Warrior collects the key and the Door appears. The Warrior enters the door, and a jumpscare image and audio plays with the intent of scaring the player. A start button also appears to return to the Start Menu.
  • Strategies: The game focuses on exploration than actual strategy.
  • Story/Setting/Premise: A Brave Warrior, after dying in battle, arrives to the door way to Heaven and meets the Guardian Angel. The Heaven in this game is based off of how kids in Christian Studies think Heaven is like. The game is more abstract than actual backstory. Because it is an exploration game, the dramatic tension comes from what the player discovers in game.
  • Target Audience: The game is for people who don't have time to play games, but does have to time to watch them.
  • Play Time: The game can be played in 30 seconds.
